# Pantallas y Funcionalidades

## Públicas
- Inicio: muestra bienvenida, tours destacados y acceso a login/registro.
- Login: permite iniciar sesión.
- Registro: permite crear cuenta.
- Lista de tours: muestra tours, filtros, búsqueda y paginación.
- Detalle del tour: muestra información completa del tour y opción de reservar.

## Cliente
- Mis reservas: muestra reservas activas y canceladas.
- Detalle de reserva: muestra la información completa de una reserva.
- Notificaciones: muestra las notificaciones del usuario.
- Perfil: muestra la información del usuario.

## Administrador
- Dashboard: muestra métricas generales.
- Gestión de tours: lista tours y permite editar o eliminar.
- Crear tour: registra un nuevo tour.
- Editar tour: actualiza información de un tour.
- Gestión de reservas: lista y filtra reservas del sistema.
