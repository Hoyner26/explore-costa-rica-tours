# Alcance del Proyecto

## Nombre del proyecto
Explore Costa Rica Tours

## Descripción
Aplicación web y móvil para explorar tours en Costa Rica, ver información detallada, reservar tours, cancelar reservas y gestionar tours como administrador.

## Usuarios
- Turista / cliente
- Administrador

## Módulos principales
- Inicio de sesión
- Tours
- Reservas
- Notificaciones
- Direcciones / enlaces a mapas
- Panel de administración

## Alcance de Fase I
- Diseño UI/UX en Figma
- Aplicación web responsive
- Aplicación móvil
- CRUD de tours
- Gestión de reservas
- Lista de notificaciones
- Login y registro
- Mock services para endpoints
- Documentación técnica
- Arquitectura de solución
- Tema claro/oscuro

## Fuera de alcance por ahora
- Backend real
- Base de datos real
- Notificaciones en tiempo real reales
- Integración con pasarela de pago
- Geolocalización en tiempo real
