# Relaciones del Sistema

- Usuario 1:N Reserva
- Tour 1:N Reserva
- Usuario 1:N Notificación

## Descripción
- Un usuario puede tener muchas reservas.
- Una reserva pertenece a un solo usuario.
- Un tour puede tener muchas reservas.
- Una reserva pertenece a un solo tour.
- Un usuario puede tener muchas notificaciones.
- Una notificación pertenece a un solo usuario.
